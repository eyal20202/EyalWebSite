# מדריך לעוזר AI

## האם API_KEY חינמי?

**לא, OpenAI API לא חינמי**, אבל:

### אפשרויות:

1. **OpenAI API (בתשלום)**:
   - GPT-4: ~$0.03 לכל 1K tokens (input) + ~$0.06 לכל 1K tokens (output)
   - GPT-3.5-turbo: הרבה יותר זול (~$0.0015 לכל 1K tokens)
   - **עלות משוערת**: כמה דולרים לחודש לשימוש בסיסי באתר אישי
   - **יתרונות**: תשובות חכמות, דינמיות, הבנה טובה של הקשר

2. **בלי API_KEY (חינמי)**:
   - ה-AI Assistant עובד עם תשובות מוכנות מראש (fallback)
   - **יתרונות**: חינמי לחלוטין, עובד מיד
   - **חסרונות**: תשובות בסיסיות, מוגבל למילות מפתח

## תשובות בסיסיות זמינות (בלי API_KEY)

העוזר יכול לענות על השאלות הבאות:

### שאלות כלליות:
- **"מי אתה?" / "מי אייל?"** - מידע על אייל
- **"מה הניסיון?"** - ניסיון תעסוקתי
- **"מה הטכנולוגיות?"** - רשימת טכנולוגיות
- **"איך ליצור קשר?"** - פרטי קשר
- **"מה הפרויקטים?"** - מידע על פרויקטים
- **"מה יש בבלוג?"** - מידע על הבלוג

### שאלות טכניות:
- **"Java" / "Spring" / "Backend"** - מידע על Backend
- **"React" / "Frontend" / "UI"** - מידע על Frontend
- **"Node.js" / "Express"** - מידע על Node.js
- **"Docker" / "Kubernetes" / "DevOps"** - מידע על DevOps
- **"Kafka"** - מידע על Kafka
- **"Redis"** - מידע על Redis

### שאלות לפי דף:
- **בדף פרויקטים**: שאלות על פרויקטים ספציפיים
- **בדף בלוג**: שאלות על מאמרים ונושאים טכניים
- **בדף משחקים**: מידע על המשחק

### שאלות נוספות:
- **"שכר" / "משכורת"** - הפניה ליצירת קשר
- **"קורות חיים" / "CV"** - מידע על קורות חיים
- **"טיפים" / "עצות"** - הפניה לבלוג
- **"קוד" / "דוגמה"** - הפניה ל-GitHub
- **"ביצועים" / "אופטימיזציה"** - מידע על אופטימיזציה
- **"בדיקות" / "testing"** - מידע על בדיקות
- **"אבטחה" / "security"** - מידע על אבטחה

## איך להפעיל AI מלא (עם API_KEY)

1. **קבל API Key מ-OpenAI**:
   - היכנס ל: https://platform.openai.com/api-keys
   - הירשם/התחבר
   - צור API key חדש
   - **עלות**: תשלם רק לפי שימוש (pay-as-you-go)

2. **הוסף לקובץ `.env`**:
   ```bash
   OPENAI_API_KEY=sk-your-api-key-here
   ```

3. **רענן את השרת**:
   ```bash
   npm run dev
   ```

## המלצה

- **לפיתוח/דמו**: השתמש ב-fallback (חינמי)
- **לייצור**: שקול להוסיף API_KEY אם אתה רוצה חוויה טובה יותר
- **עלות משוערת**: $5-20 לחודש לשימוש בסיסי באתר אישי
